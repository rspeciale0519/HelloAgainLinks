How this works technically
• On iOS: built as a native Share Extension — HAL's sheet appears without switching apps. The X app stays active underneath.
• On Android: built as an Activity with transparent/bottom-sheet theme — same overlay feel via the existing share intent.
• Tapping "Done — Back to X" dismisses the sheet and returns focus to the X app instantly.
• The native X bookmark mirror still runs in the background — so users who prefer just tapping X's own bookmark button also get their saves synced automatically.
