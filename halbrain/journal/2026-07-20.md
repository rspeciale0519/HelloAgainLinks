Audited the other tabs — blend (`/api/blends`), tags (`/api/tags`), lists (`/api/shared-lists`) use param-less GETs, no page-index bug. Evidence: only Bookmarks used pagination; pushed to develop → Codemagic build #11 (commit d1014d8) started on Mac mini M2 (Codemagic status was "Degraded" so it may run slow).
